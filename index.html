About me

Saya adalah seorang Fullstack Developer lulusan SMK Negeri 2 Surakarta yang memiliki passion dalam bidang pemrograman, khususnya dalam pengembangan aplikasi web dan mobile.

Saya memiliki pengalaman magang di Universitas Gadjah Mada (UGM), di mana saya terlibat dalam pengembangan aplikasi website untuk keperluan akademis. Selama magang, saya belajar banyak tentang praktik terbaik dalam pengembangan aplikasi serta berkolaborasi dengan tim untuk menciptakan solusi yang inovatif.

Saya telah mendapatkan sertifikasi Profesi sebagai Junior Developers dan meraih juara 3 pada perlombaan Lomba Kompetensi Siswa Kejuruhan Bidang Web Technologies Kota Surakarta Tahun 2023. Saat ini saya aktif mengembangkan berbagai proyek web dan mobile menggunakan teknologi modern.

Dengan adanya website Portfolio ini, saya berharap dapat memperkenalkan diri saya dan kemampuan saya dalam dunia pengembangan aplikasi kepada masyarakat luas.
